Week 1

Git and Node

Learning Objectives

At the end of this lesson you should have set up Git and Node.js and you will be able to:

  • Set up a Git repository and perform basic Git operations
  • Set up and use online Git repositories
  • Use Node-based modules to perform basic operations.

Download and Install Git

  • Download the Git installer for your specific operating system.
  • Follow the installation steps as you install Git using the installer.
  • Git documentation for installing Git.

Basic Git Commands

Open a cmd window or terminal on your computer. To check that Git has installed and is available:

git --version

Finish Git configuration with the following commands:

git config --global user.name "Your Name"

git config --global user.email

Check configuration with:

git config --list
